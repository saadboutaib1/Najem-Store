# Branding - Najem Store

## 1. Brand Identity

Najem Store should feel luxurious, oriental, warm, trustworthy, and refined. The visual identity should support traditional products without becoming visually heavy.

Brand keywords:

- Luxury
- Authenticity
- Oriental elegance
- Trust
- Warmth
- Clean shopping experience

## 2. Store Name

Primary name:

```text
Najem Store
```

Arabic writing can be introduced later if desired:

```text
متجر نجم
```

## 3. Brand Description

Arabic:

> Najem Store هو متجر إلكتروني متخصص في أجود أنواع العود والبخور والعطور والمِسواك، يجمع بين الأصالة والفخامة ليقدّم للزبون تجربة تسوّق سهلة، راقية وموثوقة.

English:

> Najem Store is an online store specialized in premium oud, bakhoor, perfumes and miswak, combining authenticity and luxury to offer a simple, elegant and trusted shopping experience.

| --- | --- | --- |
| Black | `#0B0B0B` | Primary background, luxury contrast |

| Beige | `#F7EFE2` | Soft backgrounds, cards, sections |
| White | `#FFFFFF` | Text on dark backgrounds, clean surfaces |
| Brown | `#6B3F1D` | Oriental warmth, secondary accents |
| Burgundy | `#7A1F1F` | Premium accent, warnings, special badges |

## 5. Typography Direction

Arabic:

- Use a readable Arabic font with elegant weight options.
- Suggested options: Cairo, Tajawal, Noto Kufi Arabic, IBM Plex Sans Arabic.

English:

- Use a clean Latin companion font.
- Suggested options: Inter, Geist, or system sans-serif.

Typography principles:

- Arabic text is the default.
- Avoid overly decorative body fonts.
- Use generous line height for Arabic readability.
- Keep product names clear and scannable.

## 7. UI Style Direction

- Default layout direction: RTL.
- Use black and beige as core surfaces.
- Use gold for premium emphasis and main actions.
- Use burgundy sparingly for special accents.
- Keep product photography large, warm, and clear.
- Use subtle borders and spacing instead of heavy shadows.
- Avoid noisy ornamental backgrounds that reduce readability.

## 8. Social Media Identity

The storefront and footer should include recognizable links/icons for:

- WhatsApp Business
- Facebook
- Instagram
- TikTok
- YouTube

The final URLs should be managed in backend store settings or environment configuration.
